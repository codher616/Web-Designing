Box

  • Content
  • Padding: Distance between content and border. Top Left Bottom Right.
  • Border: Limit of the box. Top Left Bottom Right.
    • Border: Size Type (solid, dashed, inset, outset, groove, ridge…) Color.
    • Border-radius: Curvature of the border.
  • Margin: Distance between boxes. Top Left Bottom Right. If it is auto, centers the object.
  • Box-sizing:
    • Border-box: Maintains width and height properties.
    • Content-box: Changes width and height.
  • Outline: Generates a border that does not affect the Document Object Model (DOM).

Shadow

  • Box-shadow: X-axis Y-axis Blur Border Color
  • Text-shadow: X-axis Y-axis Blur Color

Duplicate the values for more intensity

Background

  • Background-color
  • Background-image: url()
  • Background-size: size / object-fit
  • Background-repeat: repeat / no-repeat
  • Background-clip: Border-box / padding-box / content-box. Cuts the image.
  • Background-origin: Border-box / padding-box / content-box. Does not cut the image.
  • Background-position: Top / right / bottom / left / center
  • Background-atachment: Scroll / fixed

Position

  • Static: Not positioned
  • Relative: Keeps the space reserved for the box.
    • Z-index: Sorts the boxes in the z-axis (depth).
  • Absolute: Does not keep the space reserved for the box. If top/left/bottom/right are specified, it is measured from the viewport or the container (if it is positioned), not from the next boxes. Also, it adjusts the width and height to the center.
  • Fixed: Like absolute but it stays fixed
  • Sticky: Like fixed but keeps the space of the box and top or right need to be specified.

Display

  • Block: Fills the space from right to left
  • Inline: Only fills the space of the content and does not have width and height.
  • Inline-block: Like inline but has width and height
  • Flex
  • Grid

Overflow

Specifies what to do with the content that cannot enter in the box. It can be overflow-x or overflow-y.

  • Visible
  • Hidden
  • Auto
  • Scroll

Pseudo-elements

Elements that do not take part in the DOM

  • First-line/ First-letter: Changes the specifications of the first-line/letter.
  • Placeholder: Shows a text in an input until something is written. It is written in html, not in css.
  • Selection: Changes specifications of the text selected by hand.
  • After/ Before: Writes something after/before the element. The only value is content:” “.

Pseudo-classes

  • Hover: Is activated when the cursor is on the object.
  • Link: A not visited link.
  • Visited: A visited link
  • Active: Is activated when the object is being clicked.
  • Focus: Is activated when the input is clicked until the cursor is not on the object.
  • First-child / First-of-type
  • Last-child / Last-of-type
  • Nth-child () / Nth-of-type ()

Object-fit

  • Contain: Maintains the aspect ratio of the image so all the image fits in the container even if it is smaller.
  • Cover: Maintains the aspect ratio of the image, but If it is bigger than the container, the image is cut.
  • Fill: Adjusts the object aspect ratio so it fills all the container.
  • None: The object does not change its measures.
  • Scale-down: The content is dimensioned as if none or contain were specified, which would result in a smaller concrete object size.

Object-position

  • Top / Bottom
  • Right / Left
  • Px / Em / %

Flex

The flex container is divided in main axis (left to right) and cross axis (up to down). It maintains the height of the flex items by changing their width.

  • Flex-container:
    • Flex-flow (Shorthand):
      • Flex-direction:
        • Row
        • Row-reverse
        • Column
        • Column-reverse
      • Flex-wrap:
        • Nowrap
        • Wrap
        • Wrap-reverse
    • Justify-content:
      • Flex-sart
      • Flex-end
      • Center
      • Space-around: Same distance
      • Space-evenly: Auto margin in every item
      • Space-between: Biggest distance between every item
    • Align-items
      • Flex-sart
      • Flex-end
      • Flex-center
      • Stretch: Stretches the items from the top to the bottom
    • Align-content
      • Flex-sart
      • Flex-end
      • Flex-center
      • Baseline: Flex-end with wrap-reverse
  • Flex-item:
    • Margin: If it is auto put the item in the other side
    • Align-self
      • Flex-sart
      • Flex-end
      • Flex-center
      • Stretch
    • Flex (shorthand)
      • Flex-grow: Distributes the spare space depending on the number specified.
      • Flex-shrink: Specifies the space is going to give the other items when the width is less than specified. If it is 0, the item does not give space.
      • Flex-basis: Same as width but with more importance
    • Order: Defines the order by numbers

Grid

The grid container contains the grid cells, which contain a grid item and their children. Also, the grid container is formed by grid tracks (rows and columns). Finally, we can define a grid area when coding and the grid lines (column lines and row lines) are the totally lines in the grid container.

  • Grid-container:
    • Grid-template (Shorthand):
      • Grid-template-rows: Define the rows (“fr” gives the spare space).
        • Can use “repeat ()” if rows are duplicated.
          • Min-content
          • Max-content
          • Minmax (,)
          • Auto-fill: Goes up a cell, and if there are no more, generates the cells.
          • Auto-fit: Goes up a cell and fits the cells like “fr”.
      • Grid-template-columns:
        • Can use “repeat ()” if rows are duplicated.
          • Min-content
          • Max-content
          • Minmax (,)
          • Auto-fill: Goes up a cell, and if there are no more, generates the cells.
          • Auto-fit: Goes up a cell and fits the cells like “fr”.
    • Justify-content:
      • Flex-sart
      • Flex-end
      • Center
      • Space-around: Same distance
      • Space-evenly: Auto margin in every item
      • Space-between: Biggest distance between every item
    • Justify-items:
      • Flex-sart
      • Flex-end
      • Flex-center
      • Stretch: Stretches the items from the top to the bottom
    • Align-items:
      • Flex-sart
      • Flex-end
      • Flex-center
      • Stretch: Stretches the items from the top to the bottom
    • Align-content:
      • Flex-sart
      • Flex-end
      • Center
      • Space-around: Same distance
      • Space-evenly: Auto margin in every item
      • Space-between: Biggest distance between every item
  • Grid-item:
    • Grid-row (Shorthand): 1/3
      • Grid-row-start
      • Grid-row-end
    • Grid-column (Shorthand): 2/ span 5 (ocuppies 5 lines from line 2)
      • Grid-column-start
      • Grid-column-end
    • Place-self (shorthand)
      • Justify-self: Like justify-items
      • Align-self: Like align-items
    • Order: Like in flex
  • Implicit grid:
    • Grid-auto-rows
    • Grid-auto-columns
    • Grid-auto-flow: Defines how are going to be created the implicit grid cells (row or column). Also, can put “dense” so it fills the empty spaces.
  • Grid-areas
    • Grid-container (example):
      • "header header header"
      • "main main aside"
      • "main main aside"
      • "footer footer footer"
    • Grid-items:
      • Grid-area : header / main / aside / footer
  • Name a line:
    • Grid-template-rows: [line-name] 100px [line-name] 150px
